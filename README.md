# 🚀 Elon Musk - Tribute Page

Uma página de tributo dedicada a **Elon Musk**, desenvolvida como projeto prático do curso **Fullstack da B7Web**.

---

## 📋 Sobre o Projeto

Esta página apresenta uma biografia resumida de Elon Musk, destacando sua trajetória, conquistas e uma linha do tempo com os principais marcos da sua vida e carreira.

---

## 🎨 Layout

A página conta com:

- **Header** com foto de destaque e efeito de overlay
- **Seção Bio** com foto e texto descritivo
- **Timeline** com os principais eventos da vida de Elon Musk
- **Footer** com links e créditos
- Layout **responsivo** para mobile e desktop

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- Google Fonts (Bungee e Titillium Web)
- Design Responsivo com Media Queries

---

## 📚 Aprendizados

Durante o desenvolvimento deste projeto foram praticados:

- Posicionamento com CSS (position, flexbox)
- Backgrounds com gradientes e overlays
- Responsividade com media queries
- Versionamento com Git e GitHub
